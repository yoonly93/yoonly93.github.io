개인정보처리방침

최종 수정일: 2026년 5월 14일

Capa(이하 "앱")는 이용자의 개인정보를 소중히 여기며, 개인정보 보호법에 따라 아래와 같이 개인정보처리방침을 수립하여 공개합니다.

1. 수집하는 개인정보 항목

  • Firebase 익명 UID (자동 생성)
  • Firebase Installation ID (앱 설치 단위 식별자, 자동 생성)
  • 이메일 주소 (Apple/Google 소셜 로그인 시)
  • 이름 (소셜 로그인 시 제공자가 제공하는 경우)
  • 닉네임 (자동 생성 또는 직접 설정, 댓글 작성 시 표시)
  • 원두 기록 데이터 (원두명, 산지, 가공법, 평점, 테이스팅 노트 등)
  • 원두 패키지 사진 (이용자가 직접 촬영하여 업로드한 경우)
  • 레시피 데이터 (제목, 추출 단계, 음료 타입, 즐겨찾기 여부, 사용 횟수)
  • 댓글 텍스트 (이용자가 직접 작성한 경우)
  • 앱 이용 기록 (등록한 원두 수, 레시피 수, 레시피 사용 횟수 및 최근 사용 시각, 마지막 접속 시각)
  • 원두 고유 식별자 (입력된 원두 정보로부터 생성된 비식별화 해시 — 커뮤니티 집계용)
  • 앱 이용 분석 데이터 (화면 조회, 기능 사용 이벤트, 디바이스 모델, OS 버전, 언어, 국가) — Firebase Analytics 를 통해 처리
  • 광고 식별자 (AdMob 을 통한 IDFA — iOS ATT 동의 시에 한함)

2. 개인정보의 수집 및 이용 목적

  • 서비스 제공: 원두 기록 저장, 조회, 관리, 레시피 작성, 댓글 게시
  • 데이터 동기화: 기기 간 데이터 연동
  • 부정 이용 방지 및 중복 계정 식별
  • AI 기반 정보 자동 인식: 원두 패키지 사진/텍스트, 댓글·레시피 자동 다국어 번역을 위해 제3자 AI API(Anthropic Claude)로 일시 전송
  • 커뮤니티 기능: 비식별화된 원두 식별자 기반 집계, 공유 레시피 및 댓글 표시
  • 게스트 → 소셜 로그인 시 자동 데이터 마이그레이션 (옛 익명 UID 데이터의 새 계정 귀속)
  • 서비스 개선: 앱 이용 통계 분석 (Firebase Analytics)
  • 광고 제공: Google AdMob 을 통한 광고 노출 및 측정

3. 개인정보의 보유 및 이용 기간

  • 본 앱은 Firebase 익명 UID 또는 소셜 로그인(Apple/Google) 기반으로 동작합니다.
  • 회원 탈퇴 시 개인 식별 정보와 사진은 즉시 파기됩니다. 커뮤니티에 공유된 레시피·댓글·평가 데이터는 익명화하여 보존될 수 있습니다(통계·집계 무결성 유지 목적).
  • 앱을 삭제하면 기기에 저장된 데이터는 함께 삭제됩니다.
  • 익명(게스트) 상태의 서버 데이터 삭제를 원하시는 경우 아래 문의 이메일로 요청해 주시기 바랍니다.
  • 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관합니다.

4. 개인정보의 제3자 제공

  • Google Firebase: 데이터 저장, 인증, 분석(Firebase Analytics), Cloud Functions, 메시지 전송 (미국)
  • Anthropic (Claude API): 원두 패키지 사진/텍스트의 AI 분석 및 댓글·레시피 자동 번역 처리 (미국)
  • Google AdMob: 광고 제공 (미국)
  • Apple / Google: 소셜 로그인 인증 처리 (미국)
  • 상기 외 제3자에게 개인정보를 제공하지 않습니다.

5. 개인정보의 파기 절차 및 방법

  • 이용 목적이 달성된 개인정보는 지체 없이 파기합니다.
  • 전자적 파일: 복구할 수 없는 방법으로 영구 삭제
  • 기록물: 분쇄 또는 소각

6. 이용자의 권리

  • 개인정보 열람, 수정, 삭제를 요청할 수 있습니다.
  • 앱을 삭제하면 기기 내 데이터는 즉시 제거되며, 서버 데이터 삭제를 원하시면 아래 문의 이메일로 요청해 주시기 바랍니다.
  • 문의: contact@posiki.com

7. 개인정보 보호책임자

  • 이메일: contact@posiki.com

8. 본 방침의 변경

개인정보처리방침이 변경되는 경우 앱 내 공지를 통해 안내합니다.

본 개인정보처리방침은 한국어 원문을 기준으로 하며, 번역본과 차이가 있는 경우 한국어 원문이 우선합니다.
