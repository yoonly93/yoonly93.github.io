실검톡

실시간 검색어를 기반으로 같은 주제에 관심 있는 사람들이 익명으로 대화할 수 있는 한국어 채팅 서비스입니다.

개인정보처리방침 문의하기

실시간 검색어 기반

1위부터 10위까지의 검색어를 선택해 해당 주제의 대화방으로 이동합니다.

가입 없는 익명 이용

회원가입 없이 설치 단위 익명 ID와 닉네임으로 서비스를 이용합니다.

신고와 차단

메시지 신고, 사용자 차단, 운영자 숨김 처리를 통해 대화 공간을 관리합니다.

앱에서 제공하는 기능

실검 탭에서 현재 대화 가능한 검색어를 확인하고, 채팅 탭에서 내가 참여한 방을 다시 볼 수 있습니다. 종료된 방은 읽기만 가능하며 새 메시지는 작성할 수 없습니다.

  • 실검1위부터 10위까지 검색어와 채팅 진입
  • 채팅참여한 방 이력, 메시지 신고와 차단
  • 설정닉네임 수정, 알림 설정, 차단 관리

운영 원칙

실검톡은 회원가입 정보를 수집하지 않습니다. 다만 채팅과 서비스 운영을 위해 익명 사용자 ID, 닉네임, 메시지, 채팅방 정보, 신고 정보, 차단 정보, 푸시 토큰, 앱 사용 이벤트 등이 서버에 저장될 수 있습니다.

앱 삭제 후 재설치한 사용자를 기기 지문이나 광고 ID로 복구하거나 추적하지 않습니다.

운영자는 신고, 차단, 숨김 처리, 푸시 제외 키워드, 금칙어 변경을 운영 로그에 남기며, 단순 신고 수만으로 자동 제재하지 않습니다.
